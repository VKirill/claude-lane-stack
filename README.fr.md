# Claude Lane Stack

**Orchestration multi-agents de code pour Claude Code** — contrats de tâches en fichiers, lanes optionnelles AGY / Grok / Codex, ownership des chemins, board, détection de stall, **auto-merge vers `main` par le PM**.

```bash
git clone https://github.com/VKirill/claude-lane-stack.git
cd claude-lane-stack && ./install.sh
agents-doctor --apply .
claude --agent dev-orchestrator
```

Le PM est **toujours** Claude Code. Voir [README.md](README.md).
