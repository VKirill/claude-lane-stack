# Claude Lane Stack

**Orquestração multi-agente de código para Claude Code** — contratos em arquivos, lanes opcionais AGY / Grok / Codex, ownership de paths, board, detecção de stall e **auto-merge em `main` pelo PM**.

```bash
git clone https://github.com/VKirill/claude-lane-stack.git
cd claude-lane-stack && ./install.sh
agents-doctor --apply .
claude --agent dev-orchestrator
```

O PM é **sempre** Claude Code. Veja [README.md](README.md) e `docs/`.
